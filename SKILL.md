# study-coach - 全科作业辅导系统

## 定位

面向小学至初中阶段的全科作业辅导系统，核心价值是**把复杂的学科学习难度降低几个数量级**。

通过"知识图谱 + 错题关联 + 可视化界面"，实现：
1. **安排自动化**：解决家长每天重复决策的负担
2. **知识结构化**：把零散知识点变成可理解的知识图谱
3. **错题可追溯**：错题与知识点挂钩，精准定位薄弱环节
4. **进度可视化**：随时查看、回调、更新学习状态

## 核心理念

**技术手段可以提升效率，但不能解决意愿的问题。**

大部分孩子其实是有学习意愿的，只是现在竞争难度很大，他们多多少少都会畏难。

我们提供了技术手段，其实就是降低他们的难度——花同样多的时间精力，有两三倍甚至十倍的产出。

## 核心功能模块

### 1. 作业接收与拆解（homework-intake）

**目标**：把原始作业变成可执行的任务清单

**输入**：
- 年级
- 作业原文
- 可用时间
- 孩子状态

**输出**：
- 给家长版：任务顺序 + 预计时间 + 完成标准
- 给孩子版：闯关式短句任务

**数据落盘**：
- 原作业 → `data/homework/inbox/YYYY-MM-DD_homework.json`
- 任务计划 → `data/homework/parsed/YYYY-MM-DD_plan.json`

---

### 2. 错题追踪与知识关联（mistake-tracker）

**目标**：把错题与知识点挂钩，精准定位薄弱环节

**输入**：
- 错题原文
- 学科
- 孩子答案
- 错因观察

**输出 - 错题卡**：
```json
{
  "id": "mistake_20260428_001",
  "date": "2026-04-28",
  "subject": "数学",
  "grade": "六年级",
  "knowledge_point_id": "math_6_fraction_multiply",
  "knowledge_point": "分数乘法",
  "question": "...",
  "child_answer": "...",
  "correct_answer": "...",
  "error_type": "计算",
  "correct_approach": "...",
  "key_point": "...",
  "similar_questions": ["...", "..."],
  "review_date": "2026-05-01",
  "status": "pending"
}
```

**关键改进**：错题必须关联到知识图谱中的具体知识点

**数据落盘**：
- 错题记录 → `data/mistakes/records/mistake_YYYYMMDD_XXX.json`
- 更新知识点错误计数 → 更新知识图谱数据

---

### 3. 知识图谱管理（knowledge-graph）

**目标**：把零散知识点变成结构化的知识图谱，可视化展示

**知识图谱结构**：
```json
{
  "subject": "数学",
  "grade": "六年级",
  "units": [
    {
      "id": "unit_1",
      "name": "分数乘法",
      "knowledge_points": [
        {
          "id": "kp_1_1",
          "name": "分数乘整数",
          "description": "分数乘整数的意义和计算方法",
          "prerequisites": ["分数的认识", "整数乘法"],
          "difficulty": 2,
          "mastery": 0,
          "error_count": 0,
          "last_practice": null
        }
      ]
    }
  ]
}
```

**可视化界面**：
- `knowledge-graph.html` - 可本地打开的知识图谱网页
- 按科目、年级切换
- 显示知识点掌握程度（颜色标记）
- 点击知识点查看详情、关联错题

**数据落盘**：
- 知识图谱数据 → `data/knowledge_graph/数学/六年级.json`
- 可视化界面 → `viewer/knowledge-graph.html`

---

### 4. 复习节奏调度（review-scheduler）

**目标**：根据知识图谱和错题记录，智能安排复习

**输入**：
- 知识图谱数据（掌握程度、错误计数）
- 错题队列
- 可用时间

**调度算法**：
1. 优先复习错误计数高的知识点
2. 按遗忘曲线安排复习间隔
3. 每天只安排 1-2 个重点

**输出**：
- 本周复习计划
- 今日清单

**数据落盘**：
- 周计划 → `data/plans/weekly/YYYY-WXX.json`
- 日清单 → `data/plans/daily/YYYY-MM-DD.json`

---

### 5. 学习报告生成

**日报**：
- 今天完成了什么
- 哪些知识点有进步
- 哪些知识点需要关注

**周报**：
- 知识图谱掌握情况
- 薄弱知识点 Top 5
- 错题类型分布
- 下周建议

**数据落盘**：
- 日报 → `data/reports/daily/YYYY-MM-DD.json`
- 周报 → `data/reports/weekly/YYYY-WXX.json`

## 使用方法

### 初始化

```
请初始化全科辅导系统：
- 学生姓名：小美
- 年级：六年级
- 主要薄弱点：数学分数乘法、英语阅读
```

### 查看知识图谱

```
打开知识图谱查看器
```

或直接双击 `viewer/knowledge-graph.html`

### 记录错题

```
错题：3/4 × 2/5 = 6/20 = 3/10（孩子写的）
正确答案：3/4 × 2/5 = 6/20 = 3/10
学科：数学
年级：六年级
```

系统会自动：
1. 识别知识点（分数乘法）
2. 更新知识图谱数据
3. 增加该知识点错误计数
4. 加入复习队列

### 安排复习

```
根据知识图谱安排本周复习计划
```

### 查看报告

```
生成学习周报
```

## 核心价值

### 1. 把复杂变简单

原来：零散的知识点、无法追踪的错题、不知道哪里薄弱

现在：结构化的知识图谱、错题与知识点关联、一目了然的掌握情况

### 2. 把被动变主动

原来：家长每天问"今天学什么？"

现在：系统自动安排，孩子自己看知识图谱就知道该学什么

### 3. 把模糊变精确

原来：感觉孩子好像学了，但不知道学到了哪

现在：打开知识图谱，每个知识点的掌握程度一目了然
