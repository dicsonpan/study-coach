# Study Coach - 全科辅导系统

基于知识图谱的全科作业辅导系统，帮助家长精准定位孩子的学习薄弱环节。

## 核心功能

### 1. 知识图谱（Knowledge Graph）

将零散的知识点变成结构化的图谱，按科目、年级、单元、知识点分层组织。

每个知识点包含：
- **前置知识**：学这个之前，需要先掌握什么
- **后续知识**：学完这个，可以学什么
- **常见错误**：这个知识点容易错在哪
- **掌握程度**：0-100%，一目了然
- **错误计数**：关联了多少错题

### 2. 可视化界面

打开 `viewer/knowledge-graph.html` 即可使用：
- 按科目、年级切换
- 树形展示知识点层级
- 颜色标记掌握程度
- 点击知识点查看详情

### 3. 错题追踪

错题关联到知识图谱中的具体知识点，自动更新错误计数，加入复习队列。

## 使用方法

1. 克隆仓库
```bash
git clone https://github.com/dicsonpan/study-coach.git
```

2. 打开可视化界面
```bash
open viewer/knowledge-graph.html
```

3. 切换科目和年级，查看知识图谱

## 扩展知识图谱

如需添加其他年级或科目的知识图谱，欢迎：
- 提 Issue 说明需求
- 提交 Pull Request

## 技术栈

- 纯 HTML + CSS + JavaScript（可视化界面）
- JSON 数据存储（知识图谱）
- 无需后端，本地即可运行
